Question-2: What is the difference between innerHTML, innerText, and textContent ?

innnerHtml is: get all the inner html code with tag of the parent element. innerText is: get only the text of the parent element. textContent is: get only the text of the parent element.

Question-3: What is event delegation in the DOM?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

Question-4: What is event bubbling in the DOM?

Event bubbling refers to the process of propagating events from child elements to parent elements in the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

  • Create element: document.createElement('div')
  • Add element: parentNode.appendChild(element)
  • Remove element: parentNode.removeChild(element)
